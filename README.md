# Mini Biblioteka

System zarządzania biblioteką.

## Klasy
- **Ksiazka** – pojedyncza pozycja w bibliotece (tytuł, autor, liczba stron, dostępność)
- **Czytelnik** – użytkownik biblioteki (imię, nazwisko, numer karty, liczba wypożyczeń)
- **Biblioteka** – przechowuje tablicę książek, obsługuje wypożyczenia i zwroty
